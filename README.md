# 📊 A 股量化看板

一个基于 **Next.js 16 + TypeScript + Tailwind CSS** 的 A 股实时行情看板，支持基本面指标展示、智能预警、大盘指数监控和表格视图。

## 功能特性

- **实时行情** — 对接新浪财经 API，获取 A 股实时股价、涨跌幅、成交量
- **基本面指标** — 对接东方财富 API，展示市盈率、市净率、总市值、股息率、换手率、每股收益
- **大盘指数** — 上证指数、创业板指实时行情 + 恐慌指数进度条
- **安全边际评分** — 格雷厄姆估值法，0-100 评估价格是否合理
- **恐慌指数** — 基于涨跌幅/振幅/换手率的单只股票情绪指标（0 极度贪婪 → 100 极度恐慌）
- **智能预警** — 自定义规则（股息率 > 5%、换手率 > 10%、恐慌指数 > 80、成交量异常等）
- **自选股管理** — 自由添加/删除股票代码，支持多只同时监控
- **表格视图** — 数值异常自动着色（涨停红/跌停绿/高换手橙），可排序、可筛选
- **卡片视图** — 原卡片模式，看多空方向清晰
- **定时刷新** — 5/10/30/60 秒自动刷新，也可手动刷新
- **数据持久化** — 自选股、预警规则、刷新间隔、视图模式自动保存到 localStorage

## 快速开始

```bash
# 克隆项目
git clone https://github.com/LeifJG/ts-stock-monitor.git
cd ts-stock-monitor

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev
```

浏览器打开 http://localhost:3000 即可使用。

## 使用说明

1. **添加自选股** — 在顶部输入框输入 6 位股票代码（如 `600519`），点击"添加"
2. **切换视图** — 顶部右侧切换"表格/卡片"模式
3. **排序筛选** — 表格模式下点击列头排序，输入框筛选代码或名称
4. **设置预警** — 点击"预警"按钮，添加自定义规则（支持换手率、恐慌指数、成交量等）
5. **调整刷新频率** — 通过刷新控制栏选择自动刷新间隔

### 表格颜色含义

| 颜色 | 含义 |
|------|------|
| 🔴 红色 | 涨跌幅 > +9.5%（涨停） |
| 🟢 绿色 | 涨跌幅 < -9.5%（跌停） |
| 🟠 橙色加粗 | 换手率 > 10%（异常活跃） |
| 🟠 橙色 | 换手率 > 5%（较活跃） |
| 🟢 绿色加粗 | 股息率 > 5%（高股息） |
| ⚪ 灰色 | 市盈率为负（亏损） |

### 安全边际评分

基于格雷厄姆估值公式：`√(22.5 × EPS × BVPS)`

| 等级 | 评分 | 含义 |
|------|------|------|
| 🟢 优秀 | ≥ 70 | 安全边际 ≥ 30%，远低于估值 |
| 🔵 良好 | 40-70 | 安全边际 15-30%，合理偏低 |
| 🟡 一般 | 10-40 | 安全边际 0-15%，接近估值 |
| 🔴 危险 | < 10 | 高于估值，偏贵 |

### 恐慌指数

基于三个实时分量计算（无需 K 线历史）：

| 等级 | 范围 | 含义 |
|------|------|------|
| 😎 极度贪婪 | 0-20 | 市场情绪极度乐观 |
| 😊 贪婪 | 20-40 | 偏乐观 |
| 😐 中性 | 40-60 | 均衡 |
| 😰 恐慌 | 60-80 | 偏恐慌 |
| 😱 极度恐慌 | 80-100 | 市场情绪极度悲观 |

## 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **数据源**: 新浪财经 API（实时行情）、东方财富 API（基本面/换手率/EPS）

## 项目结构

```
src/
├── app/
│   ├── api/stocks/route.ts   # API 路由（个股+指数+增减持）
│   ├── globals.css            # 全局样式
│   ├── layout.tsx             # 根布局
│   └── page.tsx               # 主页面
├── components/
│   ├── AlertRuleForm.tsx      # 添加告警规则表单
│   ├── AlertRuleList.tsx      # 告警规则列表
│   ├── IndexCards.tsx         # 大盘指数卡片
│   ├── RefreshTimer.tsx       # 自动刷新控制栏
│   ├── StockCard.tsx          # 单只股票行情卡片
│   ├── StockList.tsx          # 股票卡片列表容器
│   └── StockTable.tsx         # 股票表格视图（可排序/筛选）
├── hooks/
│   ├── useAlerts.ts           # 告警规则管理（持久化）
│   ├── useLocalStorage.ts     # 通用 localStorage 持久化 Hook
│   └── useStockData.ts        # 股票数据获取 Hook
└── lib/
    ├── alert-engine.ts        # 告警规则引擎
    ├── constants.ts           # 全局常量与默认配置
    ├── indicators.ts          # 安全边际评分 & 恐慌指数计算
    ├── stock-api.ts           # 数据获取模块（新浪/东方财富适配）
    └── types.ts               # 共享类型定义
```

## 数据来源

- 实时行情：新浪财经 `https://hq.sinajs.cn/`
- 基本面/换手率/EPS：东方财富 `https://push2.eastmoney.com/`
- 增减持(预留)：东方财富数据中心

所有数据通过服务端 API 路由代理，前端不直接调用外部接口。
