# 📊 A 股量化看板

一个基于 **Next.js 16 + TypeScript + Tailwind CSS** 的 A 股实时行情看板，支持基本面指标展示、智能预警、安全边际评分和恐慌指数。

## 快速开始

```bash
git clone https://github.com/LeifJG/ts-stock-monitor.git
cd ts-stock-monitor
pnpm install
pnpm run dev
```

浏览器打开 http://localhost:3000 即可使用。

## 功能特性

### 📈 实时行情
- 对接新浪财经 API，获取 A 股实时股价、涨跌幅、成交量
- 自动刷新（5/10/30/60 秒可选），也可手动刷新
- 涨红跌绿，视觉直观

### 📊 基本面指标
- **市盈率（PE）** — 动态市盈率
- **市净率（PB）** — 市净率
- **总市值** — 流通市值/总市值
- **股息率** — 分红收益率
- **换手率** — 市场交投活跃度
- **每股收益（EPS）** — 每股盈利

数据来源：东方财富 API

### 🛡️ 安全边际评分
基于格雷厄姆估值法自动计算：
- **格雷厄姆估值** = √(22.5 × EPS × BVPS)
- **安全边际%** = （估值 - 现价）/ 估值
- **评分 0-100**，右上角徽章显示等级

| 等级 | 评分 | 含义 |
|------|------|------|
| 🟢 优秀 | ≥ 70 | 安全边际 ≥ 30%，极具吸引力 |
| 🔵 良好 | 40-70 | 安全边际 15-30%，合理偏低 |
| 🟡 一般 | 10-40 | 安全边际 0-15%，接近估值 |
| 🔴 危险 | < 10 | 低于估值，价格偏贵 |
| ⚪ 未知 | — | PE/PB 为负，不适用格雷厄姆法 |

### 😱 恐慌指数（单只股票）
综合三个分量计算，越高越恐慌：

| 分量 | 权重 | 计算方式 |
|------|:----:|---------|
| 涨跌幅 | 35% | 大跌 → 恐慌 |
| 日内振幅 | 35% | 剧烈波动 → 恐慌 |
| 换手率 | 30% | 高换手 → 恐慌 |

| 指数 | 标签 | 颜色 |
|:----:|------|:----:|
| 0-20 | 极度贪婪 😎 | 绿色 |
| 20-40 | 贪婪 😊 | 翠绿 |
| 40-60 | 中性 😐 | 黄色 |
| 60-80 | 恐慌 😰 | 橙色 |
| 80-100 | 极度恐慌 😱 | 红色 |

### 🔔 智能预警
- 自定义告警规则（支持全局规则或针对特定股票）
- 监控字段：最新价、涨跌幅、PE、PB、总市值、股息率
- 触发时卡片高亮标记，顶部横幅提示
- **持久化保存**：刷新页面不丢失

### 💾 数据持久化
- 自选股列表 → `localStorage`
- 刷新间隔 → `localStorage`
- 告警规则 → `localStorage`
- 所有配置关闭浏览器后仍然保留

## 使用说明

1. **添加自选股** — 在顶部输入框输入 6 位股票代码（如 `600519`），点击添加
2. **设置预警** — 点击右上角"预警规则"，添加自定义规则
3. **调整刷新** — 通过刷新控制栏选择自动刷新间隔
4. **查看指标** — 每只卡片展示：价格、涨跌幅、基本面、安全边际、恐慌指数

## 项目结构

```
src/
├── app/
│   ├── api/stocks/route.ts   # 后端 API 路由（代理外部数据源）
│   ├── globals.css            # 全局样式
│   ├── layout.tsx             # 根布局
│   └── page.tsx               # 主页面（组装所有组件）
├── components/
│   ├── AlertRuleForm.tsx      # 添加告警规则表单
│   ├── AlertRuleList.tsx      # 告警规则列表
│   ├── RefreshTimer.tsx       # 自动刷新控制栏
│   ├── StockCard.tsx          # 单只股票行情卡片（含安全边际 + 恐慌指数）
│   └── StockList.tsx          # 股票卡片列表容器
├── hooks/
│   ├── useAlerts.ts           # 告警规则管理 Hook（持久化）
│   ├── useLocalStorage.ts     # 通用 localStorage 持久化 Hook
│   └── useStockData.ts        # 股票数据获取 Hook
└── lib/
    ├── alert-engine.ts        # 告警规则引擎（纯函数）
    ├── constants.ts           # 全局常量与默认配置
    ├── indicators.ts          # 安全边际评分 & 恐慌指数计算
    ├── stock-api.ts           # 数据获取模块（新浪/东方财富适配）
    └── types.ts               # 共享类型定义
```

## 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **包管理**: pnpm
- **样式**: Tailwind CSS
- **持久化**: localStorage
- **数据源**: 新浪财经 API（实时行情）、东方财富 API（基本面 + 换手率 + EPS）

## 数据来源

- 实时行情：新浪财经 `https://hq.sinajs.cn/`
- 基本面：东方财富 `https://push2.eastmoney.com/`

所有数据通过服务端 API 路由代理，前端不直接调用外部接口。
